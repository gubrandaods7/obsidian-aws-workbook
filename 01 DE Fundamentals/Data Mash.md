
- Não se trata de uma tecnologia, mas sim uma forma de governança e organização dos dados.
	- Quem acessa, quem cuida, como ela é acessada

- Equipes individuais são donas dos produtos dos dados dado um determinado domínio.

- Esses produtos de dados são usados de várias formas por outras pessoas na mesma organização.
	- Exemplo: Se alguém da organização precisa de determinado dado para realizar algum tipo de análise, ela não precisa construir um processo completo de leitura/tratamento do dado, ela pode usar dessa mesma fonte que outra equipe/indivíduo gerencia.
