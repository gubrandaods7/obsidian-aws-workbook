
## 1. Visão Geral

High-level definition of data classification based on structure.

## 2. Tipos
### Estruturados
- Tabular format
- Schema enforced
- Examples: SQL databases

### Semi-Estruturados
- Flexible schema
- JSON, XML

### Não-Estruturados
- No predefined schema
- Media, text

## 3. Quando usar
- Structured → analytics, BI
- Semi → APIs, logs
- Unstructured → raw ingestion
